# Dotfiles
My linux configuration files

## Dependencies
- neovim
- git
- zsh
- Oh My Zsh
- i3-gaps
- polybar
- tmux
- alacritty

## Installation

Clone this repository and cd into it.
```bash
git clone https://github.com/forerosantiago/dotfiles.git
cd dotfiles/
```

Create the symbolic links using stow.
```bash
stow vim git```
